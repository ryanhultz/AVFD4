How to update the roster

  1. Add, edit or delete members using the buttons in the table
  2. Click Download personnel.json (or use the bar that appears at the top)
  3. Go to your GitHub repo → click personnel.json → pencil icon → paste or upload the file → Commit
  4. The run report app refreshes the roster automatically on next load

CSV import format:  Name,Badge,Qual   e.g.  Smith John,42,emt/int
Qual values: int  |  emt  |  emt/int  |  leave blank
